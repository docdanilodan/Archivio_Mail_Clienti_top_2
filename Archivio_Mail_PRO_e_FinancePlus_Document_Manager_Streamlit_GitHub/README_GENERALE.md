# Pacchetto completo - Archivio Mail Clienti PRO + FinancePlus Document Manager

Questo pacchetto contiene due web app Streamlit distinte:

1. **Archivio_Mail_Clienti_PRO**
   - archivio allegati email
   - cartelle per mittente e azienda
   - deduplica MD5
   - ricerca e report CSV
   - export ZIP

2. **FinancePlus_Document_Manager**
   - anagrafica clienti
   - fascicoli documentali
   - pratiche bancarie
   - ricerca e report
   - export ZIP

## Uso consigliato
Prima usa Archivio Mail Clienti PRO per importare/scaricare gli allegati. Poi importa i documenti rilevanti in FinancePlus Document Manager per collegarli al cliente e alla pratica bancaria.
