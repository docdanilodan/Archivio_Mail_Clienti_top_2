# Archivio Mail PRO + FinancePlus Document Manager PRO

Pacchetto Streamlit + GitHub con due web app:

1. **Archivio Mail Clienti PRO**  
   Modulo per scaricare e organizzare allegati email per periodo, mittente e azienda.

2. **FinancePlus Document Manager PRO**  
   Gestionale documentale per clienti, visure/report PDF, fascicoli, richieste, valutazioni bancarie e report PDF.

## Deploy suggerito

Crea un unico repository GitHub con tutta questa cartella.

### App 1 - Archivio Mail Clienti PRO

Main file path:

```text
Archivio_Mail_Clienti_PRO/app.py
```

### App 2 - FinancePlus Document Manager PRO

Main file path:

```text
FinancePlus_Document_Manager/app.py
```

## Novità FinancePlus Document Manager PRO

La sezione **Nuovo Cliente** e la sezione **Inserisci Report PDF** permettono di caricare un report PDF o una visura camerale e compilare automaticamente i dati dell'azienda e dell'amministratore.

Menu inclusi:

- Dashboard
- Nuovo Cliente
- Elenco Clienti
- Collaboratori
- Inserisci Report PDF
- Gestione Documenti
- Gestione Richieste
- Mail
- Valutazione bancaria
- Report PDF
- Archivio locale
- Predisposizione Google Drive
- Predisposizione pCloud
