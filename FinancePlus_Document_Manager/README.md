# FinancePlus Document Manager

Modulo Streamlit per gestione documentale clienti FinancePlus.Tech: anagrafica, fascicoli documentali, pratiche bancarie, ricerca, report, export ZIP.

## Avvio
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy
Caricare su GitHub e pubblicare su Streamlit Cloud indicando `app.py` come file principale.
